# 使用方法

对ip列表逐个ping，取响应时间最小的，作为最佳ip。

将ip地址和translate.googleapis.com拼接!! 比如 0.0.0.0 translate.googleapis.com 填写到 hosts 文件中

Windows: C:\Windows\System32\drivers\etc

MacBooks: Mac\private\etc 目录下

本人测试一般10秒内出结果。自己来感受下就知道了!

设置hosts，完记得cmd里运行一下：ipconfig/flushdns
